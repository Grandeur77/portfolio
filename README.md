# Professional Portfolio Website

A modern, responsive portfolio website showcasing professional experience, skills, projects, and career interests.

## Features

- **Responsive Design**: Fully responsive layout that works on all devices
- **Modern UI**: Clean, professional design with smooth animations
- **Smooth Scrolling**: Enhanced navigation with smooth scroll behavior
- **Interactive Elements**: Hover effects and transitions throughout
- **Mobile-Friendly**: Hamburger menu for mobile navigation
- **Easy to Customize**: Well-organized code structure for easy updates

## Sections

1. **About Me**: Professional introduction and background
2. **Professional Experience**: Detailed work experience at IFS
3. **Education**: Academic qualifications and focus areas
4. **Technical Skills**: Comprehensive list of programming languages, frameworks, tools, and technologies
5. **Projects**: Showcase of full-stack, cloud infrastructure, and mobile development projects
6. **Strengths**: Key professional strengths and capabilities
7. **Career Interests**: Areas of professional interest
8. **Professional Links**: GitHub, LinkedIn, and Email contact information

## Setup

1. Open `index.html` in a web browser
2. No build process or dependencies required - pure HTML, CSS, and JavaScript

## Customization

### Update Contact Links

1. **GitHub**: Click the GitHub link and enter your profile URL when prompted, or directly edit the `href` attribute in `index.html`
2. **LinkedIn**: Click the LinkedIn link and enter your profile URL when prompted, or directly edit the `href` attribute in `index.html`
3. **Email**: Click the email link and enter your email address when prompted, or directly edit the `href` attribute in `index.html` (line with `mailto:your.email@example.com`)

### Add Project Details

To add project names, GitHub links, and screenshots:

1. Edit the project cards in `index.html`
2. Add images in an `images` folder and reference them
3. Add GitHub links to project cards

### Color Scheme

The color scheme can be customized in `styles.css` by modifying the CSS variables in the `:root` selector:

```css
:root {
    --primary-color: #2563eb;
    --primary-dark: #1e40af;
    --secondary-color: #64748b;
    /* ... */
}
```
